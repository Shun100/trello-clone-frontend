# trello-clone-client

## 1. 環境構築

```bash
  <!-- プロジェクトの初期化 -->
  npm create vite@latest .

  <!-- 依存ライブラリのインストール -->
  npm install
  <!-- HTTP用 -->
  npm install axios
  <!-- ページ遷移制御 -->
  npm install react-router-dom
  <!-- グローバルステート管理 -->
  npm install jotai
  <!-- ドラッグ&ドロップ用 -->
  npm install @hello-pangea/dnd
```
